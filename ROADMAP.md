# Roadmap

## 0.1 MVP

- Local Markdown scan CLI.
- Human, JSON, and SARIF output.
- Risky shell, secret literal, external write, and missing confirmation rules.
- Allowlist comments.
- CI-friendly severity threshold.

## 0.2 Rule ergonomics

- Rule enable/disable toggles.
- Severity overrides.
- Expiring allowlist metadata.
- Rule documentation pages.

## 0.3 Workflow integrations

- Pre-commit hook recipe.
- GitHub Action wrapper.
- SARIF upload example.
- Agent prompt preflight examples.

## Not planned

- SaaS scanning backend.
- Hidden telemetry.
- Automatic execution of commands found in Markdown.
